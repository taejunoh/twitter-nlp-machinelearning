# TWITTER COVID-19 ANALYSIS

## Data

- Used Twitter API to extract tweets including COVID-19 words in Python
- Number of tweets: 50,000
- Period: since 12/31/2019


## Method

- Sentiment Analysis
- Topic Modeling: LDA

## Machine Learning

- LSTM with TensorFlow
- CountVectorizer with Scikit-learn
- LDA with Gensim
